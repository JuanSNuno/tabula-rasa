# Tabula Rasa - Design Thinking Project

## Directory Overview
This directory contains the documentation and research for an academic Design Thinking project named **"Tabula Rasa"**. The project conceptualizes a theoretical, highly secure, and clandestine "Identity Reestablishment" service. It explores the logistical, technological, and architectural requirements for helping individuals in extreme danger (or ultra-high-net-worth individuals) disappear completely and start a new life with a synthetic identity, bypassing both state surveillance and private threats.

The project relies on concepts like Zero-Knowledge Proofs (ZKP), Open Source Intelligence (OSINT), synthetic identity generation ("Frankenstein Identity"), and covert marketing strategies (Dog Whistling and Steganography).

## Key Files
The main content of this project is located in the `docs/` directory, which contains the following key `.docx` files:

* **`docs/Design Thinking - Reestablecimiento de identidad.docx`**
  Details the core Design Thinking methodology applied to the project (Empathize, Define, Ideate, Prototype, Evaluate). It outlines the target audience (e.g., people in mortal danger), the problem statement, the value proposition (rapid extraction, digital sanitation, new synthetic identity), and the MVP, which focuses on a secure vetting and contact system to filter out malicious actors.

* **`docs/Arquitectura Teórica y Vulnerabilidades Sistémicas en Servicios Clandestinos de Reestablecimiento de Identidad_ Proyecto _Tabula Rasa_.docx`**
  An extensive, academic-style research document exploring the theoretical architecture of the service. It analyzes historical precedents (like the Japanese *Jouhatsu*), vulnerabilities in state witness protection programs (e.g., WITSEC), and modern technological solutions for vanishing. Key topics include evasion of biometric KYC/AML controls using generative AI, exploitation of civil registry vulnerabilities, and decentralized vetting using Zero-Knowledge Proofs (zk-SNARKs).

* **`docs/Buyer Person - El Magnate sin Fronteras.docx`**
  A detailed profile of a primary target user: "The Borderless Tycoon." It describes the demographics, psychology, and behavior of an Ultra High Net Worth Individual (UHNWI) seeking to protect their wealth and family through complete disappearance, emphasizing their need for absolute confidentiality and premium, "turnkey" efficiency.

## Usage
As a non-code project, this directory is intended for reading, research, and academic review. 
* To interact with this project, review the Word documents in the `docs/` folder to understand the theoretical frameworks and design processes.
* Any future additions should follow the thematic structure of the existing documents, maintaining the academic and rigorous tone used to describe the clandestine service architecture and business model.
